---
title: 错误码
---

| code码 | 含义 | 推荐解决方法 |
| :--- | :--- | :--- |
| **通用参数错误** | | |
| 40000 | 必填参数缺失或格式不符合要求 | 检查参数名与类型 |
| 40001 | 请求的资源（如文件、项目）不存在 | 检查资源 ID 是否正确 |
| 40002 | 参数不能为空 | 补充参数 |
| 40003 | 参数为空 | 补充参数 |
| 40004 | 回调 status 值必须为 success 或 fail | 检查回调参数 |
| 40005 | 回调 origin_type 不支持 | 检查回调参数 |
| 40007 | 上传的文件类型不支持 | 仅上传允许的格式 |
| 40008 | Base64 编码内容包含非法字符 | 检查 Base64 字符串 |
| 40010 | 用户 ID 超过长度限制 | 缩短 ID 长度 |
| 40020 | Project ID 格式非法 | 检查 ID 格式 |
| 40021 | 项目不存在或已撤销 | 确认项目 ID |
| 40030 | API Key 名称过长 | 修改名称 |
| **认证与权限错误** | | |
| 40100 | 未登录或会话过期 | 请登录 |
| 40101 | 无权操作该资源 | 确认账号权限 |
| 40102 | 用户名或账号已存在 | 更换用户名 |
| 40103 | 指定的用户不存在 | 检查用户名 |
| 40110 | 验证码已过期 | 重新获取 |
| 40111 | 验证码错误 | 核对后重试 |
| 40112 | 验证码错误次数过多，锁定 10 分钟 | 稍后重试 |
| 40113 | 60秒内重复请求验证码 | 稍后重试 |
| 40114 | 单日验证码发送次数超限 | 明日再试 |
| 40130 | 请求缺少 API Key | 添加 Header |
| 40132 | API Key 无效或被禁用 | 检查 Key 状态 |
| **配额与限流错误** | | |
| 40300 | 接口调用频率超限 | <a href="/memos_cloud/limit#_4-领取更多额度" target="_blank">获取更多额度</a> |
| 40301 | Chat 请求 Token 总额超限 | <a href="/memos_cloud/limit#_4-领取更多额度" target="_blank">获取更多额度</a> |
| 40302 | Chat 响应 Token 总额超限 | <a href="/memos_cloud/limit#_4-领取更多额度" target="_blank">获取更多额度</a> |
| 40303 | 单次对话长度超过模型限制 | 缩减输入/输出长度 |
| 40304 | 账户总 API 调用次数耗尽 | <a href="/memos_cloud/limit#_4-领取更多额度" target="_blank">获取更多额度</a> |
| 40305 | 输入内容超过单次 Token 上限 | 缩减输入 |
| 40307 | 删除操作对应的记忆不存在 | 刷新或检查 ID |
| **系统与服务错误** | | |
| 50000 | 服务器内部错误 | 联系支持 |
| 50004 | 记忆服务暂时故障 | 稍后重试 |
| 50005 | 搜索服务暂时故障 | 稍后重试 |
| 50011 | SSE 建连失败 | 检查网络 |
| **知识库与文件错误** | | |
| 50100 | 文件 ID 不存在 | 确认文件未被删除 |
| 50103 | 知识库文件数量超限 | 清理或扩容 |
| 50104 | 单个文件大小超限 | 压缩文件 |
| 50105 | 知识库总容量超限 | 清理部分不需要的文档 |
| 50106 | 单文件超过 100MB | 确保小于 100MB |
| 50107 | 文件格式损坏或不匹配 | 检查文件 |
| 50108 | 知识库容量达到版本上限 | 清理部分不需要的文档 |
| 50110 | 文件不属于当前知识库 | 检查归属关系 |
| 50120 | 知识库 ID 不存在 | 检查 ID |
| 50121 | 知识库数量超限 | 考虑复用知识库 |
| 50123 | 知识库不属于当前项目 | 切换项目 |
